# Privacy Policy for Higgsfield Prompt Extender & Batch Generator

Last updated: July 17, 2026

## Data Collection

This Chrome extension does **not** collect, store, transmit, or share any personal data, browsing history, or user information.

## Local Storage Only

The extension uses Chrome's `storage.local` API solely to persist user preferences (such as sidebar width settings) locally on your device. No data is sent to any external server.

## Permissions Explained

- **`storage`**: Used only to save your sidebar width preference locally.
- **`tabs`**: Used to interact with the Higgsfield AI tab for automated batch video generation.
- **`scripting`**: Used to inject content scripts into the Higgsfield AI page for layout enhancements.
- **`sidePanel`**: Used to display the batch control dashboard as a side panel.
- **`https://higgsfield.ai/*`**: Required to access and enhance the Higgsfield AI video creation interface.

## Third-Party Services

This extension does not integrate with or send data to any third-party services, analytics platforms, or advertising networks.

## Changes to This Policy

If this policy changes, the updated version will be posted here.
